# 🚗 Arduino Nano 智能避障小车

基于 Arduino Nano 的 DIY 避障小车，使用超声波传感器与舵机云台实现前方及左右环境探测，配合前轮转向、后轮驱动的差速底盘，能够自主识别障碍物并选择最佳路径绕行。

## 🧰 所需物料

| 物料 | 数量 | 说明 |
|------|------|------|
| Arduino Nano | 1 | 主控板 |
| HC-SR04 超声波测距模块 | 1 | 测量距离 |
| SG90 舵机（或类似） | 2 | 一个用于控制前轮转向，另一个用于带动超声波传感器扫描 |
| L298N 电机驱动模块 | 1 | 驱动后轮直流电机 |
| 直流减速电机 | 2 | 后轮驱动（配合轮子） |
| 小车底盘及车轮 | 1 套 | 包括前轮转向机构（如阿克曼结构或简易万向轮） |
| 电池（7.4V 或 9V） | 1 | 为电机和控制板供电 |
| 杜邦线、电源线 | 若干 | 连接各模块 |
| 其他：螺丝、铜柱、开关等 | 适量 | 固定与扩展 |

## ⚙️ 工作原理

小车通过 **HC-SR04 超声波模块** 实时检测正前方障碍物距离。当距离小于设定阈值（如 90 cm）时，小车停车并利用 **舵机云台** 将超声波探头分别转向左、右两侧进行测距，获取左（L）、右（R）两侧的空间大小。随后根据 L 与 R 的关系执行不同的避障策略：

- **极窄空间避障**：若任一侧距离小于安全阈值（如 30 cm），先后退一段距离，再根据左右空旷程度转向并前进。
- **左右差距极小**：若左右距离相差不足 5 cm，说明两侧都较空旷，小车回正方向盘直行 2 秒，直行期间持续监测前方，一旦距离低于 55 cm 则立即后退。
- **正常避障**：选择更空旷的一侧（L > R 则左转，否则右转），转弯前会再次确认前方是否已无障碍，若已经安全则放弃转弯、直接直行。

后轮由 **L298N 模块** 驱动，速度可通过 PWM 调节。前轮转向由 **舵机** 精确控制角度，实现平滑转向。

## 🧠 代码逻辑

主程序 `loop()` 循环执行以下流程：

1. **复位与测距**：前轮回正，超声波探头指向正前方，测量前方距离。
2. **障碍判断**：若距离 < 阈值（默认 90 cm）则进入避障模式，否则全速直行。
3. **环境扫描**：探头依次转向左侧、右侧，分别记录两侧距离 L 和 R。
4. **决策分支**：
   - `L < 30 || R < 30` → 先后退 800ms，再根据 L 与 R 大小转向前进 800ms（转向角度已优化为左 45°、右 95°）。
   - `|L - R| < 5` → 回正直行 2 秒，每 100ms 监测前方，若距离 < 55cm 则紧急后退。
   - `L > R` 或 `L < R` → 转弯前再测一次前方：若已安全则回正直行，否则以较小角度（左 45° / 右 95°）转向并前进 800ms。
5. **循环**：返回步骤 1，持续自主导航。

所有阈值、速度、角度及延时均可在代码头部宏定义中便捷调整，以适应不同场地与机械结构。

## 🔌 引脚连接

| Arduino Nano | 模块引脚 |
|--------------|----------|
| D13 | HC-SR04 Trig |
| D11 | HC-SR04 Echo |
| D5  | L298N IN1 |
| D4  | L298N IN2 |
| D6  | L298N ENA (PWM) |
| D9  | 前轮转向舵机 (LR) |
| D10 | 超声波监测舵机 (CSB) |

*具体引脚可在代码中修改。*

## 🛠 调试与修改

- 串口监视器（9600 波特率）会实时打印左、右、前方距离，方便观察决策过程。
- 转向角度、速度、时间常数等均可根据你的机械结构直接修改对应变量。
- 建议先在空旷场地低速测试，逐步增大速度并微调阈值。
